# Non-Goals

Synthetic OS public documentation is not:

- the private Carter source code
- a runnable framework
- a model-routing guide
- a prompt library
- a memory database specification
- an authentication or deployment guide
- a raw log or OpRep archive
- a guarantee of correctness or safety
- a replacement for human review
- a claim of consciousness, sentience, AGI, or perfect recall
- a disclosure of proprietary governance logic
- a description of DIM as active current architecture

## What Is Not Included

This repository intentionally omits code, prompts, secrets, operational data, schemas, route names, function names, raw user data, and cloneable implementation details.
