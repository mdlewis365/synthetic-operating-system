# Sanitized Governance Trace

## Example

```text
Governance Trace

Request type: public documentation
Evidence basis: current user request plus public-safe implementation review
Memory use: context only, not authoritative proof
Retrieval use: supporting context only
Decision: produce public-safe conceptual documentation
Omissions: source code, prompt text, secrets, raw logs, raw memory, schemas
```

## What Is Not Included

This is a fictional trace. It does not expose private governance prompts, raw decision payloads, scoring logic, or implementation details.
