# Glossary

- AMS: Active Memory Subsystem. Long-term memory recall and selected memory writes.
- Carter: flagship implemented Synthetic OS runtime.
- CRM: Conversation Recovery Module. Short-term conversation continuity.
- EAS: Engineering Assistance System.
- EDR: Engineering Decision Record. Compact engineering decision summary.
- Governance Gate: deterministic EAS classifier that converts computation state and risk into bounded reporting status.
- LCM: Log Conversations Module. Selected operational logging.
- MCM: Math Computation Module. Deterministic engineering computation layer.
- OpRep: operational-report terminology for sanitized workflow reporting.
- PGM: Prompt Governance Module. Governed context assembly.
- RAG: Retrieval-Augmented Generation subsystem for non-memory knowledge retrieval.
- SIS: Synthetic Ideation System.
- Synthetic OS: private architecture pattern for governed, memory-aware AI workflows.

## What Is Not Included

The glossary does not define private prompt labels, hidden module taxonomies, internal file names, routes, schemas, credentials, or implementation-only names.
