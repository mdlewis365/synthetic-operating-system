# Roadmap

## Current Public Status

The current implementation supports Carter, SIS, EAS, memory/retrieval layers, deterministic MCM computation for engineering workflows, EDR summaries, governance classification, evidence-basis summaries, and public-safe report sanitation.

## Public Documentation Roadmap

- Keep public docs synchronized with implemented behavior.
- Expand sanitized examples without using private logs or user data.
- Add clearer public diagrams as the architecture stabilizes.
- Document new modules only after implementation is verified.
- Keep private prompt and governance internals out of public repositories.

## What Is Not Included

This roadmap does not disclose private engineering backlog, proprietary module designs, deployment plans, credentials, source code, prompt drafts, model-routing plans, or unimplemented architecture as current fact.
